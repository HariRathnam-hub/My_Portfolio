# Hari Rathnam — Personal Portfolio

My personal developer portfolio built with vanilla HTML, CSS, and JavaScript. Hosted on GitHub Pages.

🔗 **Live Site:** [harirathnam-hub.github.io/My_Portfolio](https://harirathnam-hub.github.io/My_Portfolio/)

---

## Built With

- **HTML5 / CSS3** — no frameworks, no build step
- **Space Grotesk** — display & body typeface
- **JetBrains Mono** — monospace accent typeface
- **Tabler Icons** — outline icon set

## Running Locally

No build step needed — just open the file:

```bash
git clone https://github.com/HariRathnam-hub/My_Portfolio.git
cd My_Portfolio
open index.html   # or double-click the file
```

---

## Updating Content

| What to change | Where |
|----------------|-------|
| Name / tagline / bio | `index.html` → `.hero` section |
| Add a project | Copy a `.project-card` block in the `#projects` section |
| Add a cert | Copy a `.cert-item` block in the `#certifications` section |
| Update resume | Replace the PDF file and update the `href` in the hero CTA |
| Social links | Update `href` values on `.social-row` anchors and in `#contact` |
